## 一、文档目的

本文档定义投研分析系统的数据可视化需求，用于指导UI设计和前后端开发。

核心原则：
- **纯价值导向**：数据展示服务于投资决策，不做无意义的装饰
- **信息密度优先**：在有限空间内最大化信息传递效率
- **风险可视化**：流动性风险必须直观可感知

## 三、图表类型规格

### 3.1 数字卡片
- **用途：** 单指标当前值展示
- **要素：** 当前值、对上期变化（数值+百分比）、趋势箭头
- **颜色：** 正向变化绿色，负向变化红色
- **尺寸：** 固定宽度 120px，高度自适应

### 3.2 折线图
- **用途：** 时序趋势展示
- **要素：** X轴时间轴、Y轴数值、Tooltip悬浮信息、图例
- **交互：** 支持时间范围选择、缩放、导出
- **标注：** 重要拐点需标注数值

### 3.3 仪表盘/进度环
- **用途：** 单指标达成度/状态展示
- **要素：** 中心数值、环形进度条、等级刻度
- **颜色段：** 红色(0-40%)、黄色(40-70%)、绿色(70-100%)

### 3.4 面积图
- **用途：** 规模累积展示（如美联储资产负债表）
- **要素：** 面积填充、X轴时间、Y轴数值
- **颜色：** 半透明渐变填充

### 3.5 环形图
- **用途：** 周期/分类占比展示
- **要素：** 中心标签、环形占比、图例
- **交互：** 点击扇区展开详情

### 3.6 分位数区间图
- **用途：** 当前估值在历史中的位置
- **要素：** 迷你箱线图、当前值标记点、历史区间带
- **标注：** 标注25th、50th、75th分位线

### 3.7 堆叠柱状图
- **用途：** 多指标对比（如社融/M1/M2剪刀差）
- **要素：** X轴分类、Y轴数值、正负区分色

### 3.8 热力图
- **用途：** 行业/板块估值对比
- **要素：** 行列矩阵、颜色渐变映射数值、Tooltip
- **颜色：** 低估值绿色，高估值红色

## 四、页面布局需求

### 4.1 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│ 顶部导航栏                                                    │
├─────────────────────────────────────────────────────────────┤
│ 流动性监控面板（固定顶部，始终可见）                              │
│ [US10Y] [CN10Y] [利差] [USD/CNY] [LPR] [剪刀差]              │
├─────────────────────────────────────────────────────────────┤
│ 风险状态栏（触发预警时显示）                                    │
├──────────────────────┬──────────────────────────────────────┤
│                      │                                       │
│   骨（宏观经济）      │     筋（资产周期）                     │
│   4格数字卡片        │     环形图+状态标签                   │
│                      │                                       │
├──────────────────────┼──────────────────────────────────────┤
│                      │                                       │
│   肉（政策面）        │     个股分析入口                      │
│   政策卡片+时间轴     │     搜索框+快捷入口                   │
│                      │                                       │
└──────────────────────┴──────────────────────────────────────┘
```

### 4.2 流动性监控面板（必须固定顶部）

```
┌─────────┬─────────┬─────────┬─────────┬─────────┬─────────┐
│ US10Y   │ CN10Y   │ 中美利差 │ USD/CNY │ LPR 1Y  │ 剪刀差   │
│ 4.52%   │ 2.31%   │ 221BP  │ 7.2450  │ 3.45%   │ -2.1%   │
│ ↑+5BP   │ ↓-3BP   │ 警戒线  │ 正常     │ 不变     │ 需关注   │
└─────────┴─────────┴─────────┴─────────┴─────────┴─────────┘
```

### 4.3 风险熔断提示（弹窗）

当触发风险条件时：
- 全屏遮罩（半透明红色）
- 中央弹窗显示预警信息
- 包含：触发条件、当前数值、建议操作
- 需人工确认后方可关闭

## 五、交互需求

### 5.1 时间维度控制
- 全局时间范围选择器（支持：日/周/月/季/年切换）
- 自定义时间范围选择（开始日期-结束日期）
- 时间轴支持拖拽平移

### 5.2 筛选与排序
- 多条件筛选：行业、市值区间、估值区间、ROE区间
- 自定义排序：按市值/ROE/安全边际/PE等
- 筛选条件保存为预设

### 5.3 详情展开
- 点击数字卡片 → 弹出详情面板（展示历史数据表格+图表）
- 点击图表元素 → 显示详细Tooltip
- 右键快捷菜单：导出数据、复制数值、添加到自选

### 5.4 联动高亮
- 悬停时显示十字准线
- 同指标多图表联动高亮
- 相关性高的指标自动关联（如US10Y与CN10Y利差）

### 5.5 数据操作
- 数据导出为CSV/Excel
- 图表导出为PNG/SVG
- 报告生成（PDF格式）

## 六、数据来源与更新策略

### 6.1 数据源优先级

| 优先级 | 数据类型 | 推荐来源 | 备选来源 |
|--------|----------|----------|----------|
| P0 | 宏观数据 | 国家统计局API | wind/同花顺 |
| P0 | 利率数据 | 外汇交易中心 | 中国货币网 |
| P0 | 汇率数据 | 中国货币网 | 外汇交易中心 |
| P0 | 国债数据 | 中债估值中心 | 外汇交易中心 |
| P1 | 美联储数据 | FRED API | 美联储官网 |
| P1 | 股市行情 | 行情服务商API | 交易所API |
| P1 | 财务数据 | 财报数据 | 财务数据库 |

### 6.2 更新频率

| 数据类型 | 更新频率 | 缓存策略 |
|----------|----------|----------|
| 利率/汇率/国债 | 日度(T+0或T+1) | 5分钟 |
| 宏观数据 | 月度(滞后2周) | 24小时 |
| 美联储数据 | 周度 | 1小时 |
| 财务数据 | 季报后更新 | 24小时 |

### 6.3 离线缓存
- 需支持至少最近24个月的历史数据缓存
- 支持无网络状态下的历史数据查看

## 七、非功能性需求

### 7.1 性能要求
- 首屏加载时间 < 3秒
- 图表渲染时间 < 1秒
- 数据刷新不影响用户操作

### 7.2 响应式要求
- 支持桌面端（1920px+）
- 支持平板端（768px-1919px）
- 移动端暂不支持（投研场景以桌面为主）

### 7.3 辅助功能
- 深色模式支持
- 数据播报功能（语音朗读关键指标）
- 快捷键支持（K线图常用快捷键）

## 八、附录：指标计算公式

### 8.1 安全边际
```
安全边际 = (内在价值 - 当前股价) / 内在价值 × 100%
```

### 8.2 中美利差
```
中美利差 = CN10Y收益率 - US10Y收益率
```

### 8.3 剪刀差
```
剪刀差 = 社融增速 - M1增速 或 社融增速 - M2增速
```

### 8.4 净利润现金含量
```
净利润现金含量 = 经营现金流净额 / 净利润 × 100%
```
