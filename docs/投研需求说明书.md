## 一、文档目的

本文档定义投研分析系统的核心需求，用于指导UI设计和前后端开发。

核心原则：
- **纯价值导向**：数据展示服务于投资决策
- **流动性前置**：宏观流动性分析是入场的前置条件
- **风险可视化**：流动性风险必须直观可感知

---

## 二、骨-筋-肉-血框架

### 2.1 四维体系概述

| 维度 | 定义 | 核心作用 |
|------|------|----------|
| 骨 | 宏观经济 | 决定市场整体运行环境 |
| 筋 | 资产周期 | 决定大类资产配置时钟 |
| 肉 | 政策面 | 决定行业短期盈利弹性 |
| 血 | 流动性 | 决定估值中枢和风险偏好 |

**分析优先级：血 → 骨 → 筋 → 肉**

### 2.2 骨（宏观经济）

**核心指标：**

| 指标名称 | 数据类型 | 更新频率 | 图表类型 | 优先级 |
|----------|----------|----------|----------|--------|
| CPI同比/环比 | 百分比 | 月度 | 数字卡片+折线图 | P0 |
| PPI同比/环比 | 百分比 | 月度 | 数字卡片+折线图 | P0 |
| 城镇调查失业率 | 百分比 | 月度 | 数字卡片+折线图 | P0 |
| 官方PMI / 财新PMI | 数值(0-100) | 月度 | 仪表盘+折线图 | P0 |

**分析逻辑：**
```
CPI高位 + PPI低位 → 滞胀象限（防御为主）
CPI低位 + PPI回暖 → 复苏前期（积极布局）
PMI > 50 + 趋势向上 → 扩张期（顺周期）
PMI < 50 + 趋势向下 → 收缩期（防御）
```

### 2.3 筋（资产周期）

**核心指标：**

| 指标名称 | 数据类型 | 更新频率 | 图表类型 | 优先级 |
|----------|----------|----------|----------|--------|
| 美林时钟阶段 | 枚举(复苏/过热/滞胀/衰退) | 季度 | 环形图+状态标签 | P0 |
| 库存周期位置 | 枚举(去库/补库) | 月度 | 环形图+状态标签 | P0 |
| 南华综合指数 | 数值 | 日度 | 折线图 | P1 |

**美林时钟配置建议：**
- 复苏：股票优先，债券标配
- 过热：商品优先，债券低配
- 滞胀：现金高配，股票低配
- 衰退：债券优先，股票低配

### 2.4 肉（政策面）

**核心指标：**

| 指标名称 | 数据类型 | 更新频率 | 图表类型 | 优先级 |
|----------|----------|----------|----------|--------|
| 财政赤字率 | 百分比 | 月度 | 数字卡片+折线图 | P0 |
| 产业政策列表 | 结构化文本 | 按需 | 政策卡片列表 | P0 |
| 重要会议时间轴 | 日期+内容 | 按需 | 水平时间轴 | P1 |

### 2.5 血（流动性）- 核心监控区域

**外部流动性（全球视角）：**

| 指标名称 | 数据类型 | 更新频率 | 图表类型 | 优先级 |
|----------|----------|----------|----------|--------|
| 美联储资产负债表 | 万亿美元 | 周度 | 面积图 | P0 |
| DXY美元指数 | 数值 | 日度 | 折线图 | P0 |
| US10Y收益率 | 百分比 | 日度 | 折线图 | P0 |

**内部流动性（国内核心）：**

| 指标名称 | 数据类型 | 更新频率 | 图表类型 | 优先级 |
|----------|----------|----------|----------|--------|
| USD/CNY在岸/离岸 | 数值 | 日度 | 折线图 | P0 |
| CN10Y收益率 | 百分比 | 日度 | 折线图 | P0 |
| 中美利差 | 基点 | 日度 | 折线图 | P0 |
| LPR 1年期/5年期 | 百分比 | 月度 | 仪表盘 | P0 |
| SHIBOR(O/N,3M,1Y) | 百分比 | 日度 | 折线图 | P1 |
| 社融存量增速 | 百分比 | 月度 | 折线图 | P0 |
| M1/M2增速 | 百分比 | 月度 | 折线图 | P0 |
| 社融/M1/M2剪刀差 | 差值 | 月度 | 柱状图 | P0 |

**风险熔断条件（满足任一触发）：**
1. 中美利差倒挂 < -100BP 伴随USD/CNY单周贬值 > 2%
2. 社融当月增量同比下滑 > 30%
3. SHIBOR O/N突然飙升 > 5%
4. 央行公开市场操作连续净回笼 > 4周

## 三、数据可视化需求

### 3.1 页面布局

**流动性监控面板（必须固定顶部）：**
```
┌─────────┬─────────┬─────────┬─────────┬─────────┬─────────┐
│ US10Y   │ CN10Y   │ 中美利差 │ USD/CNY │ LPR 1Y  │ 剪刀差   │
│ 4.52%   │ 2.31%   │ 221BP  │ 7.2450  │ 3.45%   │ -2.1%   │
│ ↑+5BP   │ ↓-3BP   │ 警戒线  │ 正常     │ 不变     │ 需关注   │
└─────────┴─────────┴─────────┴─────────┴─────────┴─────────┘
```

**整体架构：**
```
┌─────────────────────────────────────────────────────────────┐
│ 顶部导航栏                                                    │
├─────────────────────────────────────────────────────────────┤
│ 流动性监控面板（固定顶部，始终可见）                              │
├─────────────────────────────────────────────────────────────┤
│ 风险状态栏（触发预警时显示）                                    │
├──────────────────────┬──────────────────────────────────────┤
│   骨（宏观经济）      │     筋（资产周期）                       │
│   4格数字卡片        │     环形图+状态标签                       │
├──────────────────────┼──────────────────────────────────────┤
│   肉（政策面）        │     个股分析入口                        │
│   政策卡片+时间轴     │     搜索框+快捷入口                      │
└──────────────────────┴──────────────────────────────────────┘
```

### 3.2 图表类型建议

| 场景 | 推荐图表 | 说明 |
|------|----------|------|
| 单指标状态 | 仪表盘/数字卡片 | 直观显示当前值 |
| 时序趋势 | 折线图 | 时间轴数据展示 |
| 多指标对比 | 分组柱状图 | 同周期多指标对比 |
| 占比分布 | 环形图/饼图 | 组成结构展示 |
| 风险预警 | 颜色渐变热力图 | 高风险区域高亮 |
| 历史分位 | 分位数区间图 | 当前估值历史位置 |
| 规模累积 | 面积图 | 美联储资产负债表等 |
| 多指标差值 | 堆叠柱状图 | 剪刀差等 |

### 3.3 交互需求

1. **时间维度**
   - 全局时间范围选择器（日/周/月/季/年切换）
   - 自定义时间范围选择
   - 时间轴支持拖拽平移

2. **筛选与排序**
   - 多条件筛选（行业、市值、估值区间）
   - 自定义排序
   - 筛选条件保存为预设

3. **详情展开**
   - 点击数字卡片 → 弹出详情面板
   - 点击图表元素 → 显示Tooltip
   - 右键快捷菜单：导出数据、复制数值

4. **联动高亮**
   - 悬停时显示十字准线
   - 相关性高的指标自动关联

5. **风险熔断提示**
   - 全屏遮罩（半透明红色）
   - 中央弹窗显示预警信息
   - 需人工确认后方可关闭

## 五、附录：关键计算公式

### 5.1 中美利差
```
中美利差 = CN10Y收益率 - US10Y收益率
```

### 5.2 剪刀差
```
剪刀差 = 社融增速 - M1增速 或 社融增速 - M2增速
```

### 5.3 净利润现金含量
```
净利润现金含量 = 经营现金流净额 / 净利润 × 100%
```
