# 估值模型需求

**版本：** v1.0
**作者：** 投研顾问
**日期：** 2026-05-10

---

## 一、概述

本规格定义投研分析系统的估值模型，用于后端计算逻辑实现。

**核心原则：** 根据企业生命周期匹配估值模型，将CN10Y国债收益率作为无风险利率（Rf）代入计算。

---

## 二、模型选择矩阵

| 企业生命周期 | 典型行业 | 推荐模型 | 备选模型 |
|--------------|----------|----------|----------|
| 成熟期/高股息 | 银行、保险、水电、铁路 | DDM、RIM、PB-ROE | DCF |
| 成熟期/现金牛 | 消费、公用事业、白酒 | DCF (FCFF/FCFE) | PE/PB |
| 成长期 | 科技、医药、新能源 | PE、PEG | PS、EV/S |
| 重资产/并购 | 地产、钢铁、煤炭 | EV/EBITDA | PB、RNAV |

---

## 三、DDM（股息折现模型）

### 3.1 适用场景
高股息、盈利稳定的成熟企业（银行、水电）

### 3.2 公式
```
V = Σ(Dt / (1+Ke)^t) + Dn / (Ke - g) / (1+Ke)^n

其中:
- V = 内在价值
- Dt = 第t期预期股息
- Ke = 权益成本 = Rf + β × ERP
- Rf = 无风险利率（使用CN10Y）
- g = 永续增长率（2%-3%）
- n = 预测期（5-10年）
```

### 3.3 参数需求

| 参数 | 说明 | 来源/范围 |
|------|------|-----------|
| Rf | 无风险利率 | 动态获取CN10Y |
| ERP | 股权风险溢价 | 5.5%（中国市场） |
| β | Beta系数 | 行业参考值 |
| g | 永续增长率 | 2%-3% |
| 预测期 | 股息预测年数 | 5-10年 |
| 预期股息 | 各年股息预测 | 公司历史+指引 |

### 3.4 输出
- 每股内在价值（元/股）
- 隐含收益率（股息率 + 增长率）

---

## 四、DCF（现金流折现模型）

### 4.1 适用场景
现金流稳定、自由现金流充沛的成熟企业

### 4.2 公式
```
FCFF模型:
V = Σ(FCFFt / (1+WACC)^t) + FCFFn × (1+g) / (WACC - g) / (1+WACC)^n

WACC = Ke × E/(E+D) + Kd × (1-T) × D/(E+D)
```

### 4.3 参数需求

| 参数 | 说明 | 来源 |
|------|------|------|
| FCFF | 自由现金流 | 财报计算 |
| 营收增长率 | 未来5年营收CAGR | 行业均值/公司指引 |
| 毛利率趋势 | 毛利率变化假设 | 历史均值 |
| WACC | 加权平均资本成本 | 计算得出 |
| g | 永续增长率 | 2%-3% |

### 4.4 输出
- 每股内在价值
- 自由现金流收益率

---

## 五、PE / PEG（市盈率模型）

### 5.1 适用场景
盈利稳定的成长期企业

### 5.2 公式
```
PE = P / EPS
PEG = PE / G = (P / EPS) / G
```

### 5.3 参数需求

| 参数 | 说明 | 来源 |
|------|------|------|
| 当前PE | 市盈率 | 实时行情 |
| 历史PE分位 | 过去5年分位数 | 计算得出 |
| 预期G | 未来3年净利润CAGR | 券商一致预期或自行调整 |

### 5.4 安全边际判定
- PEG < 1 → 相对低估
- PEG = 1 → 合理估值
- PEG > 1 → 相对高估

### 5.5 输出
- 当前PE
- 历史分位数
- PEG值
- 预期合理股价

---

## 六、EV/EBITDA

### 6.1 适用场景
重资产、资本密集型企业（地产、钢铁、煤炭）

### 6.2 公式
```
EV = EBITDA × Multiple
EV = 市值 + 净负债

其中:
- Multiple = 行业可比公司中位数
```

### 6.3 参数需求

| 参数 | 说明 | 来源 |
|------|------|------|
| EBITDA | 息税折旧摊销前利润 | 财报计算 |
| 净负债 | 有息负债 - 现金 | 资产负债表 |
| Comparable Multiple | 可比公司EV/EBITDA中位数 | 行业数据 |

### 6.4 输出
- EV估值
- 每股内在价值
- 相对估值分位

---

## 七、PB-ROE（市净率模型）

### 7.1 适用场景
银行、保险等使用净资产计价的企业

### 7.2 公式
```
合理PB = (ROE - g) / (Ke - g)
价值PB = 合理PB × 安全系数（0.6-0.8）

其中:
- Ke = Rf + β × ERP
- g = 永续增长率
```

### 7.3 参数需求

| 参数 | 说明 | 来源 |
|------|------|------|
| ROE | 净资产收益率 | 财报 |
| BVPS | 每股净资产 | 资产负债表 |
| Rf | 无风险利率 | CN10Y |
| ERP | 股权风险溢价 | 5.5% |
| β | Beta系数 | 行业参考 |
| g | 永续增长率 | 2%-3% |

### 7.4 输出
- 当前PB
- 合理PB
- 价值PB
- 内在价值

---

## 八、RIM（剩余收益模型）

### 8.1 适用场景
高盈利质量、账面价值稳定的企业

### 8.2 公式
```
V = BV + Σ(ROEt - Ke) × BVt-1 / (1+Ke)^t

其中:
- BV = 每股净资产
- ROEt = 预期第t期ROE
- Ke = 权益成本
```

---

## 九、无风险利率（Rf）应用

### 9.1 Rf来源
```
Rf = CN10Y十年期国债收益率（最新值）
```
- 数据来源：中国外汇交易中心/中债估值中心
- 更新频率：每日

### 9.2 Rf在估值中的影响

| 模型 | Rf应用位置 | 影响 |
|------|------------|------|
| DDM | Ke = Rf + β × ERP | Rf↑ → Ke↑ → V↓ |
| DCF | WACC中的Ke | Rf↑ → WACC↑ → V↓ |
| PB-ROE | Ke = Rf + β × ERP | Rf↑ → Ke↑ → 合理PB↓ |
| RIM | Ke = Rf + β × ERP | Rf↑ → Ke↑ → V↓ |

### 9.3 Rf动态获取需求
- 系统定时获取CN10Y最新收益率
- Rf变更时触发重估提醒
- 支持手动输入Rf进行敏感性分析

## 十、安全边际计算

### 10.1 公式
```
安全边际 = (内在价值 - 当前股价) / 内在价值 × 100%
```

### 10.2 判定规则

| 安全边际 | 判定 | 建议操作 |
|----------|------|----------|
| > 30% | 深度低估 | 重仓买入 |
| 20%-30% | 偏低估 | 重点买入 |
| 10%-20% | 合理偏低 | 标配观望 |
| 0%-10% | 合理偏高 | 低配 |
| < 0% | 高估 | 规避/卖出 |

---

## 十一、数据输入需求

### 11.1 必需输入数据

| 数据类型 | 字段 | 格式 |
|----------|------|------|
| 价格数据 | 当前股价 | 实时 |
| 财务数据 | 营收、净利润、FCF | 年度（近5年） |
| 财务数据 | 毛利率、净利率 | 年度（近5年） |
| 财务数据 | ROE、ROIC | 年度（近5年） |
| 财务数据 | 每股净资产（BVPS） | 最新 |
| 财务数据 | 总股本、流通股本 | 最新 |
| 财务数据 | 有息负债、现金及等价物 | 最新 |
| 市场数据 | Beta系数 | 行业参考 |
| 市场数据 | 历史PE/PB分位 | 计算 |

### 11.2 数据校验规则
- 净利润与经营现金流差异 > 50% → 提示盈利质量风险
- 负ROE → 禁用DDM/RIM，改用EV/EBITDA
- Beta < 0.5 或 > 2 → 提示异常值

## 十二、系统功能需求

### 12.1 模型计算
- 支持一键自动选择模型
- 支持手动指定模型
- 支持多模型并行计算
- 输出最优模型估值（取多模型中位数或保守值）

### 12.2 敏感性分析
- Rf（无风险利率）± 50BP
- 增长率（g）± 1%
- WACC ± 100BP
- 输出Tornado Chart

### 12.3 历史回测
- 记录历史估值结果
- 支持历史安全边际追踪
- 生成估值偏离度报告

### 12.4 批量估值
- 支持上传股票池批量计算
- 输出批量估值结果表格
- 支持按安全边际排序筛选

### 12.5 报告生成
- 生成个股估值报告（PDF）
- 支持导出Excel数据
