## 一、技术方案选型

### 1.1 技术栈全景

```
┌─────────────────────────────────────────────────────────────┐
│                        核心框架                              │
│  Next.js 16 (App Router) + TypeScript + Tailwind CSS 4      │
├─────────────────────────────────────────────────────────────┤
│                        UI 组件                               │
│  shadcn/ui + Radix UI + class-variance-authority            │
├─────────────────────────────────────────────────────────────┤
│                        图表/可视化                            │
│  Recharts + Tremor (数据可视化专用)                          │
├─────────────────────────────────────────────────────────────┤
│                        状态管理                              │
│  Zustand (轻量) + React Query / SWR (服务端状态)            │
├─────────────────────────────────────────────────────────────┤
│                        样式方案                              │
│  Tailwind CSS 4 + CSS Variables (设计系统)                   │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 选型理由

| 类别 | 选型 | 理由 |
|------|------|------|
| **框架** | Next.js 16 App Router | 支持 RSC、Server Actions、API Routes 一体化 |
| **类型** | TypeScript strict | 投研数据计算需要类型安全 |
| **样式** | Tailwind CSS 4 | 与 shadcn/ui 完美集成，支持 CSS 变量主题 |
| **组件** | shadcn/ui | 可定制、设计一致、维护成本低 |
| **图表** | Recharts | React 原生、tree-shaking 友好、社区活跃 |
| **状态** | Zustand | 比 Redux 轻量，比 Context 灵活 |
| **数据** | React Query / SWR | 缓存、预取、乐观更新 |

### 5.2 API 客户端封装

```typescript
// lib/api/client.ts
class ApiClient {
  private baseUrl = '/api'

  async get<T>(endpoint: string, params?: Record<string, string>) {
    const url = new URL(`${this.baseUrl}${endpoint}`, window.location.origin)
    if (params) {
      Object.entries(params).forEach(([k, v]) => url.searchParams.set(k, v))
    }

    const response = await fetch(url.toString())
    if (!response.ok) throw new ApiError(response.status)

    return response.json() as Promise<ApiResponse<T>>
  }

  async post<T>(endpoint: string, body: unknown) {
    const response = await fetch(`${this.baseUrl}${endpoint}`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(body),
    })

    if (!response.ok) throw new ApiError(response.status)
    return response.json() as Promise<ApiResponse<T>>
  }
}
```

### 5.4 数据获取 Hooks

```typescript
// lib/hooks/useMacroData.ts
export function useMacroData(timeframe: Timeframe, dateRange: DateRange) {
  return useQuery({
    queryKey: ['macro', timeframe, dateRange],
    queryFn: () => api.get<MacroSnapshot[]>('/macro/snapshots', {
      timeframe,
      startDate: dateRange.start.toISOString(),
      endDate: dateRange.end.toISOString(),
    }),
    staleTime: 5 * 60 * 1000,  // 5分钟
  })
}

// lib/hooks/useStockAnalysis.ts
export function useStockAnalysis(code: string) {
  return useQuery({
    queryKey: ['stock', code, 'analysis'],
    queryFn: () => api.get<StockAnalysis>(`/stock/${code}/analysis`),
  })
}
```

### 7.3 数据缓存策略

| 数据类型 | 缓存时间 | 刷新策略 |
|----------|----------|----------|
| 实时数据（汇率/国债） | 5 分钟 | 后台自动刷新 |
