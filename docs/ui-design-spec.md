**版本：** v1.0

**设计师：** UI设计师

---

## 1. 设计理念

### 1.1 产品定位
专业投研数据可视化平台，服务于价值投资分析师。设计风格需传达"专业、精准、冷静"的核心气质。

### 1.2 视觉基调
- **主色调：** Cyber Finance 深空风格（暗色系）
- **强调色：** 电光蓝 #3b82f6（Primary）、霓虹橙 #f97316（Accent）
- **情绪传达：** 专业严肃但不冰冷，数据密集但有呼吸感

### 1.3 设计原则
1. **信息优先：** 数据展示 > 装饰元素
2. **层次清晰：** 核心指标 > 辅助指标 > 详情
3. **交互克制：** 避免过度动效干扰分析决策
4. **响应迅速：** 支持实时数据刷新

## 2. 页面结构

### 2.1 整体架构
```
┌─────────────────────────────────────────────────────────┐
│ Header（顶部导航 + Logo + 快捷操作）                      │
├──────────┬──────────────────────────────────────────────┤
│          │                                              │
│ Sidebar  │              Main Content                   │
│ (左侧    │              (主内容区)                      │
│ 导航)    │                                              │
│          │                                              │
└──────────┴──────────────────────────────────────────────┘
```

### 2.2 页面规划

| 页面 | 功能 | 核心组件 |
|------|------|----------|
| **Dashboard 首页** | 四象限概览 + 流动性警报 | MetricCard、AlertBanner |
| **宏观分析（骨）** | CPI/PPI、失业率、PMI 趋势图 | LineChart、DataTable |
| **周期分析（筋）** | 美林时钟、库存周期仪表盘 | GaugeChart、DonutChart |
| **政策分析（肉）** | 财政赤字、产业扶持政策 | BarChart、PolicyCard |
| **流动性分析（血）** | 汇率、国债、SHIBOR 多线图 | MultiLineChart、Heatmap |
| **个股分析** | 估值仪表盘、安全边际指示器 | ValuationGauge、ComparisonTable |
| **个股详情** | 深度财务分析 + 历史估值 | FinancialCharts、RadarChart |

## 5. 组件规范

### 5.1 指标卡片 MetricCard
```
┌────────────────────┐
│ Label (Micro)      │
│                    │
│ Value (Metric)     │
│                    │
│ SubLabel (Caption) │
└────────────────────┘
```
- **尺寸：** 固定 160x120px 或响应式
- **状态：** Default / Hover（发光边框）/ Loading（骨架）
- **变体：** 按维度区分边框色（骨蓝/筋橙/肉紫/血绿）

### 5.2 安全边际仪表盘 ValuationGauge
```
     ╭─────────────╮
    ╱               ╲
   │   ┌───────┐    │
   │   │  28%  │    │  ← 指针指示当前值
   │   │ 安全边际│    │
   │   └───────┘    │
    ╲               ╱
     ╰─────────────╯
    低估   合理   高估
```
- **刻度：** 0% - 30% - 50% - 100%
- **颜色分段：** 0-15%红色 / 15-30%黄色 / 30%+绿色
- **动画：** 指针过渡 700ms ease-out

### 5.3 美林时钟仪表盘 MerrillGauge
```
        复苏 ●───┐
                │
    ┌───────────┴───────────┐
    │                       │
    │    ╭─────────────╮    │
    │   ╱               ╲   │
    │  │    当前: 过热   │   │  ← 环形进度指示
    │   ╲               ╱   │
    │    ╰─────────────╯    │
    │                       │
    └───────────┬───────────┘
                │
        滞胀 ●──┘
```
- **四个阶段：** 复苏/过热/滞胀/衰退
- **当前指示：** 发光圆点 + 颜色高亮
- **悬停效果：** 显示该阶段特征说明

### 5.4 数据表格 DataTable
```
┌─────────────────────────────────────────┐
│ 表头（可排序 ↑↓）                           │
├──────────┬──────────┬──────────┬─────────┤
│ 指标名称  │ 最新值   │ 环比变化 │ 趋势图   │
├──────────┼──────────┼──────────┼─────────┤
│ CN10Y    │ 2.85%    │ +0.05    │ ▁▂▃▅   │
│ USD/CNY  │ 7.2456   │ -0.012   │ ▅▃▂▁   │
└──────────┴──────────┴──────────┴─────────┘
```
- **行高：** 48px
- **悬停：** 背景变为 --bg-hover
- **排序：** 点击切换升/降序
- **固定列：** 左侧指标名称列 sticky

### 5.5 趋势折线图 LineChart
```
↑
│    ╭─╮           ╭───╮
│ ───╯ ╰───╮   ╭──╯     ╰───
│           ╰──╯
│
└──────────────────────────────────→
  1月   2月   3月   4月   5月
```
- **线条：** 2px 宽度，实线
- **点：** 4px 圆点，悬停放大至 8px
- **区域：** 半透明填充（opacity 0.2）
- **网格：** 横向虚线，颜色 rgba(255,255,255,0.05)
- **缩放：** 支持时间范围选择器

- **交互：** 悬停显示详情 tooltip

## 6. 交互规范

### 6.1 数据刷新策略

| 数据类型 | 更新频率 | 视觉反馈 |
|----------|----------|----------|
| 实时数据（汇率/国债） | WebSocket 推送 | 数值闪烁 + 颜色变化 |
| 日度数据（宏观指标） | 每日 09:30 更新 | 时间戳刷新 + Badge |
| 周度数据（政策） | 周一更新 | 版本标识 |

### 6.2 数据源标注
```
┌─────────────────────┐
│ 官方 ● 估算 ○ 预测 ◎ │
└─────────────────────┘
```
- **● 实心圆：** 官方数据（国家统计局等）
- **○ 空心圆：** 计算估算值
- **◎ 圆环：** 预测值

### 6.3 状态反馈

| 状态 | 视觉表现 |
|------|----------|
| Loading | 骨架动画 shimmer |
| Error | 红色边框 + 错误图标 + 重试按钮 |
| Empty | 占位图 + "暂无数据"文案 |
| Success | 绿色勾选 + Toast 提示 |

## 8. 无障碍规范

- **对比度：** 文本与背景对比度 ≥ 4.5:1
- **焦点可见：** 键盘导航时显示焦点轮廓
- **ARIA 标签：** 图表需添加 aria-label 描述
- **色盲友好：** 不依赖颜色传递单一信息

## 9. 技术实现建议

### 9.1 推荐图表库
- **主选：** Recharts（React 友好，tree-shaking）
- **备选：** Tremor（数据可视化组件库，与 shadcn 风格统一）

### 9.2 关键组件映射
```
设计稿 → shadcn/ui 组件
─────────────────────────────────
指标卡片 → Card + 自定义样式
数据表格 → Table + 排序功能
标签切换 → Tabs
表单输入 → Input + Select
弹窗详情 → Dialog
快捷操作 → DropdownMenu
消息提示 → Toast
```

### 9.3 性能优化
- 图表懒加载：Intersection Observer
- 数据缓存：SWR / React Query
- 虚拟列表：大数据表格使用 react-virtual
