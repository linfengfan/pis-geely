## 一、设计风格定位

### 1.1 产品定位

专业投研数据可视化平台，服务于价值投资分析师。

### 1.2 视觉基调

| 属性 | 描述 |
|------|------|
| **风格** | Cyber Finance 深空风格（暗色系） |
| **情绪** | 专业严肃但不冰冷，数据密集但有呼吸感 |
| **核心** | 信息优先、层次清晰、交互克制、响应迅速 |

### 1.3 设计原则

1. **信息优先** - 数据展示 > 装饰元素
2. **层次清晰** - 核心指标 > 辅助指标 > 详情
3. **交互克制** - 避免过度动效干扰分析决策
4. **响应迅速** - 支持实时数据刷新

### 1.3.1 已确认需求（v1.3）

| 项目 | 确认状态 | 说明 |
|------|----------|------|
| 多语言 | ❌ 不需要 | 首期只支持中文，术语（PMI/PE/DCF）保持原样 |
| 深色/亮色模式 | ✅ 必须支持 | 默认暗色模式，切换入口在顶部导航栏右侧 |
| 数据导出 | ✅ 必须支持 | Excel(CSV/XLSX)、PDF报告、图表(PNG/SVG) |
| 移动端 | ❌ 不支持 | 桌面端为主(1920px+)，平板端考虑基础布局 |
| 数据源颜色 | ✅ 已纳入 | 官方(绿)、估算(黄)、预测(蓝) |
| 安全边际刻度 | ✅ 已更新 | 0-10%(红)、10-30%(黄)、>30%(绿) |

3. 图表需支持时间轴切换（日/周/月/季/年）
4. 深色模式支持（默认）+ 可选亮色模式

### 4.2 核心组件规范

#### 4.2.1 流动性监控面板 LiquidityMonitorPanel（关键组件，固定顶部）

```
┌─────────┬─────────┬─────────┬─────────┬─────────┬─────────┐
│ US10Y   │ CN10Y   │ 中美利差 │ USD/CNY │ LPR 1Y  │ 剪刀差   │
│ 4.52%   │ 2.31%   │ 221BP   │ 7.2450  │ 3.45%   │ -2.1%   │
│ ↑+5BP   │ ↓-3BP   │ ⚠️警戒  │ 正常    │ 不变    │ 需关注   │
└─────────┴─────────┴─────────┴─────────┴─────────┴─────────┘
```

**规格：**
- 位置：固定顶部（position: sticky），始终可见
- 高度：60-80px
- 背景：var(--bg-surface) + 底部阴影
- 展示6个核心指标：US10Y、CN10Y、中美利差、USD/CNY、LPR 1Y、剪刀差

**状态：**
- 正常：绿色文字
- 警示：橙色文字 + 警告图标
- 危险：红色文字 + 脉冲动画

**交互：**
- 点击任一指标 → 跳转至流动性详情页
- 实时数据自动更新（WebSocket）

**规格：**
- 尺寸：600px × 500px（居中）
- 边框：3px solid var(--color-danger)
- 背景遮罩：rgba(239, 68, 68, 0.15)

**动画（关键约束）：**
- 边框脉冲动画：@keyframes pulse-border { 0%, 100% { box-shadow: 0 0 20px rgba(239, 68, 68, 0.8); } 50% { box-shadow: 0 0 40px rgba(239, 68, 68, 1); } }

**交互：**
- 全屏半透明遮罩，不可点击穿透
- 必须点击"已知悉"按钮关闭

**触发条件：**
- 中美利差 > 150BP 持续 5 个交易日 → 红色预警
- USD/CNY 单日贬值 > 1% → 橙色预警
- 社融当月增量同比下滑 > 30% → 红色预警

## 七、无障碍规范

- **对比度**：文本与背景对比度 ≥ 4.5:1
- **焦点可见**：键盘导航时显示焦点轮廓
- **ARIA 标签**：图表需添加 aria-label 描述
- **色盲友好**：不依赖颜色传递单一信息

## 八、技术实现建议

### 8.1 推荐图表库

- **主选**：Recharts（React 友好，tree-shaking）
- **备选**：Tremor（数据可视化组件库，与 shadcn 风格统一）

### 8.2 特殊图表实现

| 图表 | 实现方式 |
|------|----------|
| 安全边际仪表盘 | SVG 环形进度 + 自定义动画 |
| 美林时钟 | SVG 自定义环形图 |
| 流动性热力图 | Canvas 或 SVG 自定义实现 |

### 8.3 性能优化

- 图表懒加载：Intersection Observer
- 数据缓存：SWR / React Query
- 虚拟列表：大数据表格使用 react-virtual
- 实时数据：WebSocket
